#Megabonk
Un juego donde tienes que eliminar enemigos (cuadrados) con mas o menos vida, con diferentes habilidades y grandaria, que te intentarán vencer. 
Por cada oleada, aparecen mas y mas dificiles, pero cada vez que cambias de oleada tendrás una nueva habilidad. 
El juego contiene una tienda en la que, si tu compras mejoras con las monedas que te dan los enemigos al matarlos; puedes mejorar tanto el alcance de tu arma (un martillo), la fuerza, rapidez, alcance... etc.
Para jugar tienes que entrar en https://juanito777890.github.io/Megabonk/Megabonk.html
